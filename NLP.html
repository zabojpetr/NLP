1. Entropie textu¶

Každý cyklus¶

Spotat $P(i,j)$ - pravdpodobnost vskytu dvojice $i,j$ v textu
Spotat $P(j|i)$ - pravdpodobnost vskytu $j$, kdy se vyskytlo $i$
Spotat $H(J|I) = - \sum_{i \in I, j \in J}P(i,j)log_2 P(j|i)$
Spotat $PX(P(J|I)) = 2^{H(J|I)}$

def char_mess_up(seed, mess_prob, file):
    original_data = get_data(file)
    original_chars = np.unique(np.concatenate([list(x) for x in original_data]))
    rnd = random.Random()
    rnd.seed(seed)
    
    mess_up_data = original_data.copy()
    for i in range(mess_up_data.size):
        l = list(mess_up_data[i])
        for j in range(len(l)):
            if rnd.random() <= mess_prob:
                l[j] = original_chars[rnd.randrange(original_chars.size)]
        mess_up_data[i] = ''.join(l)
    
    
    if(DEBUG):
        print("Metoda: char_mess_up")
        print("Seed: ",seed)
        print("Pravdpodobnost zmny: ",mess_prob)
        print("Soubor: ", file)
        print("Pvodn data: ", original_data)
        print("Znaky: ", original_chars)
        print("Vsledek: ", mess_up_data)
        print()
        
    
    return mess_up_data

Výpočet entropie¶

Pro data (list slov) spotme entropii. Nejprve spotme etnosti slov $c(i)$ (odeteme 1 od etnosti poslednho slova, protoe netvo dvojici), kter budeme ukldat do slovnku i:c(i). Pot spotme etnosti dvojic $c(i,j)$, kter ulome tak do slovnku (i,j):c(i,j).
$P(i,j) = \dfrac{c(i,j)}{N-1}$, kde $N$ je velikost dat.
$P(j|i) = \dfrac{c(i,j)}{\sum_j c(i,j)} = \dfrac{c(i,j)}{c(i)}$

def entropy(data):
    
    data = np.concatenate((["<s>"],data))
    # Odest 1 od etnosti poslednho slova, protoe to u nem dvojici a j chci z toho potat psti
    words, counts = np.unique(data, return_counts=True)
    counts_words = dict(zip(words,counts))
    
    counts_words[data[-1]] -= 1   
    
    counts_pairs = dict()
    for i in range(data.size - 1):
        if (data[i],data[i+1]) in counts_pairs:
            counts_pairs[(data[i],data[i+1])] += 1
        else:
            counts_pairs[(data[i],data[i+1])] = 1
    
    H = 0
    N = data.size
    for pair in counts_pairs:
        H -= counts_pairs[pair]/(N-1)*math.log2(counts_pairs[pair]/counts_words[pair[0]])
    
    PX = math.pow(2,H)
    
    return (H,PX)

def get_info(data):
#     results = np.zeros(shape=(len(INFO_INDEX),len(FILES)), dtype=np.dtype('U16'))
    original_data = data
    results = np.array([original_data.size], dtype=np.dtype('U16'))
    
    results = np.append(results, np.concatenate([list(x) for x in original_data]).size)
    
    
    original_words, counts_words = np.unique(original_data, return_counts=True)
    results = np.append(results, original_words.size)
    
    originalChars = np.unique(np.concatenate([list(x) for x in original_data]))
    results = np.append(results, originalChars.size)
    
    results = np.append(results, "{0:.2f}".format(np.concatenate([list(x) for x in original_data]).size/original_data.size))
    
    results = np.append(results, "{0:.2f}".format(np.concatenate([list(x) for x in original_words]).size/original_words.size))
    
    results = np.append(results, "{0:.2f}".format(np.sum(counts_words)/original_words.size))
    
    c = counts_words.copy()
    w = original_words.copy()
    w = np.delete(w,np.argmax(counts_words))
    c = np.delete(c,np.argmax(counts_words))
    c2 = c.copy()
    w2 = w.copy()
    w2 = np.delete(w,np.argmax(c))
    c2 = np.delete(c,np.argmax(c))
    results = np.append(results, str(np.max(counts_words)) + " [" + original_words[np.argmax(counts_words)] + "]")
    results = np.append(results, str(np.max(c)) + " [" + w[np.argmax(c)] + "]")
    results = np.append(results, str(np.max(c2)) + " [" + w2[np.argmax(c2)] + "]")
    
    results = np.append(results, np.min(counts_words))
    
    results = np.append(results, sum([1 for x in counts_words if x == np.min(counts_words)]))
    
    results = np.append(results, sum([1 for x in original_data if len(x)==1]))
    
    results = np.append(results, sum([1 for x in original_words if len(x)==1]))
        
    return results

Entropie po náhodných změnách znaků¶

Pi nhodn zmn znak dochz k vytven novch slov. To znamen, e pibv slov, kter se vyskytnou pouze 1. Tedy se entropie sniuje.
Pi zmn slova $i$ na $i'$, pokud vznikne nov slovo $$-\dfrac{c(i,j)}{N-1}log_2 \dfrac{c(i,j)}{c(i)} \geq -\dfrac{c(i,j)-1}{N-1}log_2 \dfrac{c(i,j)-1}{c(i)-1}-0$$ Nerovnost plat dky tomu, e $1 \leq c(i,j) \leq c(i)$
Pi zmn slova $i$ na $i'$, pokud nevznikne nov slovo $$-\dfrac{c(i,j)}{N-1}log_2 \dfrac{c(i,j)}{c(i)}-\dfrac{c(i',j)}{N-1}log_2 \dfrac{c(i',j)}{c(i')} \stackrel{?}{\lesseqgtr} -\dfrac{c(i,j)-1}{N-1}log_2 \dfrac{c(i,j)-1}{c(i)-1}-\dfrac{c(i',j)+1}{N-1}log_2 \dfrac{c(i',j)+1}{c(i')+1}$$ Zde u nen odpov zcela jasn. Mohou nastat vechny ppady.

c(i,j) c(i) c(i',j) c(i') Nerovnost
2 1000 2 1000 <
3 1000 2 1000 >
2 10 2 1000 >
2 1000 2 10 <

Z tabulky si piblin dokeme pedstavit, co se pi zmnch dje. Protoe esk text obsahuje 2 vce jednopsmennch slov, a prv u tchto slov nevznik nov slovo, astji se bude entropie zvyovat. Z grafu tak vidme, e entropie eskho textu kles pomaleji.

Entropie po náhodných změnách slov¶

Pi tto zmn nedochz k vzniku novch slov, pouze se upravuj jejich etnosti, tomu odpovd druh vzorec uveden ve. Pi nhodn zmn slov se v podstat vyrovnvaj etnosti.

Myslm si, e dvodem snen entropie eskho textu je vlastnost logaritmu a vysok poet slov s etnost 1. Pili o 11 000 slov s etnost 1, bu ztrtou slova (maximln 1 500 slov) , a nebo zvenm etnosti (cca 9 500 slov). U prvnho ppadu entropie roste a u druhho kles dky vlastnosti logaritmu.

U anglickho textu je to obdobn, ale protoe poet slov s malou etnost je mnohem mn, mohlo dojt k nrstu entropie.

Psací úloha¶

Co vme: $$H_1 = H_2 = E$$ $$E = H_1 = -\sum \dfrac{c(i,j)}{N}log_2\dfrac{c(i,j)}{c(i)}$$ $$N*E = -\sum c(i,j)log_2\dfrac{c(i,j)}{c(i)}$$ $$E = H_2 = -\sum \dfrac{c(u,v)}{M}log_2\dfrac{c(u,v)}{c(u)}$$ $$M*E = -\sum c(u,v)log_2\dfrac{c(u,v)}{c(u)}$$ $$E \stackrel{?}{\lesseqgtr} -\sum \dfrac{c(x,y)}{M+N+1}log_2\dfrac{c(x,y)}{c(x)}$$ $$(M+N+1)E \stackrel{?}{\lesseqgtr} -\sum c(x,y)log_2\dfrac{c(x,y)}{c(x)}$$ Nyn rozepeme sumu novho textu na entropie $T_1$, $T_2$ a spojovac dvojice: $$ -\sum c(x,y)log_2\dfrac{c(x,y)}{c(x)} = -\sum c(i,j)log_2\dfrac{c(i,j)}{c(i)} -\sum c(u,v)log_2\dfrac{c(u,v)}{c(u)} - 1log_2 \dfrac{1}{c(i')} = N*E + M*E - log_2 \dfrac{1}{c(i')}$$ kde $i'$ je posledn slovo v textu $T_1$. Jeliko o hodnot $-log_2 \dfrac{1}{c(i')}$ vme pouze to, e je vt rovna $0$, nememe porovnat ob strany nerovnice,protoe neznme porovnn k hodnote $E$. Tedy mohou nastat vechny nerovnost (i rovnost).

Entropie spojenho textu, a j neumme urit pesn, bude piblin E, protoe jak jsme si ukzali, zvis pouze na spojovacm lenu.
